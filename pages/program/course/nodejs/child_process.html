child_process（子进程）

spawn 

let ls = require('child_process').spawn(command:string, args:Array<string>, options:Object);

例如： 

require('child_process').spawn("node", ['./demo.js'], {
    stdio: 'inherit'
});

等价于： 

node ./demo.js

也就是运行命令 “node” 并携带参数 “./demo.js”。

回调钩子

close

命令执行完毕后触发： 

ls.on('close', () => {
    // todo
});
